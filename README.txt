PASOS PARA DESPLEGAR TU BOT EN RENDER

1. Crear nuevo servicio en https://render.com
2. Subir este paquete como GitHub repo o manualmente en "Manual Deploy"
3. Configurar variables de entorno desde el archivo .env (incluido)
4. Backend: Python FastAPI | Frontend: React (panel)
5. Conectar tu bot de Telegram si querés señales manuales
6. ¡Listo! Tu bot va a estar operando 24/7

Si tenés dudas, podés volver a este chat para ayuda adicional.
